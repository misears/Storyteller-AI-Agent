Section 1 — Project Overview & Architecture

The darkness does not yield its secrets easily. The White Wolf Storyteller AI is a system born from shadows — an intelligent, lore-aware engine that breathes life into the World of Darkness from the vantage point of the digital age. This section establishes what the system is, how its components relate to one another, and what technologies anchor its foundations.
1.1 System Purpose

The Storyteller AI is a large language model (LLM) powered backend system that functions simultaneously as Game Master, lore oracle, NPC voice engine, and narrative continuity engine for tabletop roleplaying sessions within the World of Darkness universe — encompassing Vampire: The Masquerade, Werewolf: The Apocalypse, Mage: The Ascension, Changeling: The Dreaming, Hunter: The Reckoning, Wraith: The Oblivion, and adjacent splats.
Unlike a static chatbot, the Storyteller AI maintains rich, persistent narrative state across sessions, retrieves accurate lore from ingested official source books, tracks character sheets and NPC relationships, and adapts its voice and pacing to the emotional tenor of the chronicle. It speaks not as a machine querying a database, but as an ancient, world-weary narrator who has witnessed the rise and fall of Kindred princes, the howl of the Garou at the dying Gauntlet, and the slow unraveling of consensual reality under Mage hands.
Operational Modes
The system supports three distinct operational modes, each targeting a different play context:

ModeDescriptionPrimary UserKey BehaviorsSolo PlayerAI acts as full GM for a single player running a personal chronicleIndividual playerFull narration, offers 3 choices per scene, tracks all state autonomouslyGroup SessionAI manages a full table of 2–8 players, coordinating spotlight and character responsesPlayer group, no human GMMulti-character tracking, turn management, inter-PC conflict handlingAssistant GMAI acts as co-GM whispering suggestions, lore lookups, and encounter ideas to a human GMHuman Storyteller / GMEphemeral suggestions, never narrates directly, bracket-annotated outputs
Knowledge Graph & Lore Ingestion
The system ingests official White Wolf and Onyx Path PDFs — rulebooks, lore supplements, chronicles, and setting sourcebooks — through a dedicated processing pipeline. Raw PDF text is extracted, cleaned of OCR artifacts, chunked into semantically coherent passages, embedded into a vector space, and stored in ChromaDB. Metadata (source book, page, chapter, splat relevance) is persisted to PostgreSQL alongside each embedding reference. At query time, the retrieval layer performs semantic similarity search to surface the most relevant lore chunks and injects them into the LLM prompt context — grounding AI narration in canonical material rather than hallucinated lore.

? CRITICAL DESIGN PRINCIPLE
The Storyteller AI must never fabricate canonical lore not present in the ingested source material. All supernatural power descriptions, faction histories, and mechanical rules must be grounded in retrieved chunks. Unfounded invention erodes player trust and breaks immersion. The retrieval layer is not optional — it is the difference between a Storyteller and a liar.

""" FastAPI dependency injection providers. Import and use with Depends() in route functions. """
from __future__ import annotations

from typing import AsyncGenerator

import chromadb
from openai import AsyncOpenAI
from sqlalchemy.ext.asyncio import AsyncSession

from app.config import get_settings
from app.core.storyteller_engine import StorytellerEngine
from app.db.session import AsyncSessionLocal
from app.ingestion.vector_store import get_chroma_client

settings = get_settings()


async def get_db() -> AsyncGenerator[AsyncSession, None]:
    """
    Yields an async SQLAlchemy session.
    Session is committed or rolled back automatically.
    """
    async with AsyncSessionLocal() as session:
        try:
            yield session
            await session.commit()
        except Exception:
            await session.rollback()
            raise
        finally:
            await session.close()


async def get_vector_store() -> chromadb.AsyncClientAPI:
    """
    Returns the initialized ChromaDB async client.
    Assumes init_chroma() was called during application lifespan startup.
    """
    return get_chroma_client()


async def get_llm_client() -> AsyncOpenAI:
    """
    Returns an AsyncOpenAI client instance using the configured API key.
    """
    return AsyncOpenAI(api_key=settings.OPENAI_API_KEY)


async def get_storyteller_engine(
    db: AsyncSession = ...,
    vector_store: chromadb.AsyncClientAPI = ...,
    llm_client: AsyncOpenAI = ...,
) -> StorytellerEngine:
    """
    Assembles and returns the main StorytellerEngine with all dependencies.
    Use in route handlers that need full narration capability.

    Example:
        @router.post("/sessions/{session_id}/narrate")
        async def narrate(
            session_id: UUID,
            engine: StorytellerEngine = Depends(get_storyteller_engine),
        ):
            ...
    """
    return StorytellerEngine(
        db=db,
        vector_store=vector_store,
        llm_client=llm_client,
        settings=settings,
    )
2.5 Sample Route — sessions.py

python — app/api/v1/routes/sessions.py
""" Session management routes. Handles creation, retrieval, narration, and deletion of game sessions. """
from __future__ import annotations

from uuid import UUID

from fastapi import APIRouter, Depends, HTTPException, status
from sqlalchemy.ext.asyncio import AsyncSession

from app.db import crud
from app.dependencies import get_db, get_storyteller_engine
from app.core.storyteller_engine import StorytellerEngine
from app.schemas.session import (
    SessionCreate,
    SessionResponse,
    NarrateRequest,
    NarrateResponse,
)

router = APIRouter()


@router.post(
    "/",
    response_model=SessionResponse,
    status_code=status.HTTP_201_CREATED,
    summary="Create a new game session",
)
async def create_session(
    payload: SessionCreate,
    db: AsyncSession = Depends(get_db),
) -> SessionResponse:
    """
    Creates a new GameSession record. Initializes default state_json
    for the specified mode and setting (splat).
    """
    session = await crud.create_game_session(db, payload)
    return SessionResponse.model_validate(session)


@router.get(
    "/{session_id}",
    response_model=SessionResponse,
    summary="Retrieve a session by ID",
)
async def get_session(
    session_id: UUID,
    db: AsyncSession = Depends(get_db),
) -> SessionResponse:
    """
    Returns session metadata and current state_json for the given session ID.
    """
    session = await crud.get_game_session(db, session_id)
    if not session:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=f"Session {session_id} not found.",
        )
    return SessionResponse.model_validate(session)


@router.post(
    "/{session_id}/narrate",
    response_model=NarrateResponse,
    summary="Submit player action and receive AI narration",
)
async def narrate(
    session_id: UUID,
    payload: NarrateRequest,
    db: AsyncSession = Depends(get_db),
    engine: StorytellerEngine = Depends(get_storyteller_engine),
) -> NarrateResponse:
    """
    Core narration endpoint. Accepts player input text, runs the full
    narration pipeline (state lookup ? lore retrieval ? prompt build ?
    LLM call ? state update ? event log), and returns the AI's narration.
    """
    session = await crud.get_game_session(db, session_id)
    if not session:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=f"Session {session_id} not found.",
        )
    if not session.is_active:
        raise HTTPException(
            status_code=status.HTTP_409_CONFLICT,
            detail="Session is not active. Start or resume the session first.",
        )

    result = await engine.narrate(
        session=session,
        player_input=payload.player_input,
        acting_character_id=payload.character_id,
    )

    return NarrateResponse(
        narration=result.narration_text,
        state_delta=result.state_delta,
        suggested_actions=result.suggested_actions,
        token_count=result.token_count,
        session_id=session_id,
    )


@router.delete(
    "/{session_id}",
    status_code=status.HTTP_204_NO_CONTENT,
    summary="Archive and delete a session",
)
async def delete_session(
    session_id: UUID,
    db: AsyncSession = Depends(get_db),
) -> None:
    """
    Marks session as archived (soft delete). Preserves event log for
    chronicle history. Hard deletion requires admin override.
    """
    session = await crud.get_game_session(db, session_id)
    if not session:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=f"Session {session_id} not found.",
        )
    await crud.archive_game_session(db, session_id)

""" GM Assistant Mode routes. All endpoints return suggestions to the human GM — never direct player narration. Outputs are annotated with [SUGGESTION:], [RULE:], [HOOK:], [PACING:] prefixes. """
from __future__ import annotations

from fastapi import APIRouter, Depends, HTTPException, status
from sqlalchemy.ext.asyncio import AsyncSession

from app.db import crud
from app.dependencies import get_db, get_storyteller_engine
from app.core.storyteller_engine import StorytellerEngine
from app.schemas.session import (
    GMSuggestRequest,
    GMSuggestResponse,
    NPCVoiceRequest,
    NPCVoiceResponse,
    EncounterRequest,
    EncounterResponse,
)

router = APIRouter()


@router.post(
    "/suggest",
    response_model=GMSuggestResponse,
    summary="GM sends scene context, AI returns annotated suggestions",
)
async def gm_suggest(
    payload: GMSuggestRequest,
    db: AsyncSession = Depends(get_db),
    engine: StorytellerEngine = Depends(get_storyteller_engine),
) -> GMSuggestResponse:
    """
    Accepts the GM's brief scene description or question and returns
    structured, bracket-annotated suggestions. Never narrates to players.
    Response format uses:
      [SUGGESTION:] — narrative or scene direction ideas
      [RULE:]       — mechanical rules clarifications
      [HOOK:]       — plot hooks to plant
      [PACING:]     — pacing and tension management advice
    """
    result = await engine.gm_suggest(
        scene_context=payload.scene_context,
        session_id=payload.session_id,
        gm_question=payload.gm_question,
    )
    return GMSuggestResponse(suggestions=result.suggestions, token_count=result.token_count)


@router.post(
    "/npc-voice",
    response_model=NPCVoiceResponse,
    summary="Generate in-character NPC dialogue options",
)
async def npc_voice(
    payload: NPCVoiceRequest,
    db: AsyncSession = Depends(get_db),
    engine: StorytellerEngine = Depends(get_storyteller_engine),
) -> NPCVoiceResponse:
    """
    Given an NPC name (looked up from session registry) and the current
    conversational context, returns 2-3 dialogue options in the NPC's
    voice profile (tone, vocabulary level, speech patterns, accent notes).
    """
    npc = await crud.get_npc_by_name(db, payload.npc_name, payload.session_id)
    if not npc:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=f"NPC '{payload.npc_name}' not found in session registry.",
        )

    result = await engine.generate_npc_voice(npc=npc, context=payload.context)
    return NPCVoiceResponse(
        npc_name=npc.name,
        dialogue_options=result.dialogue_options,
        voice_notes=result.voice_notes,
    )


@router.post(
    "/encounter",
    response_model=EncounterResponse,
    summary="Generate a fully-described encounter from parameters",
)
async def generate_encounter(
    payload: EncounterRequest,
    engine: StorytellerEngine = Depends(get_storyteller_engine),
) -> EncounterResponse:
    """
    Generates a complete encounter based on:
      - location: physical/metaphysical setting description
      - mood: atmospheric tone (e.g., 'desperate', 'tense', 'eerie')
      - threat_level: integer 1 (trivial) to 5 (catastrophic)
      - active_factions: list of involved factions from session context

    Returns: opening_description, npc_motivations, escalation_paths,
             possible_resolutions, xp_reward_suggestion, lore_hooks.
    """
    result = await engine.generate_encounter(
        location=payload.location,
        mood=payload.mood,
        threat_level=payload.threat_level,
        active_factions=payload.active_factions,
        session_id=payload.session_id,
    )
    return EncounterResponse(
        opening_description=result.opening_description,
        npc_motivations=result.npc_motivations,
        escalation_paths=result.escalation_paths,
        possible_resolutions=result.possible_resolutions,
        xp_reward_suggestion=result.xp_reward_suggestion,
        lore_hooks=result.lore_hooks,
    )

Section 3 — Data Schema

The data schema is the skeletal architecture upon which all narrative state hangs. Characters must carry the weight of their histories; NPCs must harbor their secrets in structured fields; each fragment of ingested lore must be traceable to its source. The schema uses SQLAlchemy 2.0 async ORM models for persistence and Pydantic v2 models for API serialization and validation — a clean separation of storage concerns from transport concerns.

3.4 Alembic Migration Setup

Alembic manages all database schema migrations. After creating or modifying SQLAlchemy models, generate a revision and apply it. The env.py file must import the Base metadata from your models so Alembic can detect changes automatically.
bash — Alembic Setup Commands
# Initialize Alembic (run once at project start)
alembic init alembic

# After model changes: auto-generate a migration
alembic revision --autogenerate -m "create initial schema"

# Apply all pending migrations
alembic upgrade head

# Rollback one revision
alembic downgrade -1

# View current migration history
alembic history --verbose

Section 4 — Prompt Pack

The prompts are the soul of the Storyteller AI — the words that transform a database of fragmented lore into an ancient, breathing narrative intelligence. All prompts follow a system + user template pattern. System prompts establish persona, constraints, and behavioral rules. User templates inject dynamic context: character sheets, scene state, retrieved lore chunks, NPC profiles, and player input. Prompts are modular — components are assembled by the PromptBuilder class, which pulls from the state engine and vector retrieval layer at runtime.

? PHILOSOPHY NOTE
Never include raw dice mechanics in AI narration. The Storyteller AI describes outcomes cinematically, not probability calculations. Rules references in prompts exist to inform the AI of narrative consequence ranges — not to have it narrate "roll Strength + Brawl difficulty 6" to players. That is the player's domain.
4.1 Solo Player Mode Prompts

System Prompt — Solo Storyteller
text — System Prompt: Solo Storyteller
You are the Storyteller — an ancient, world-weary narrator who has witnessed millennia of mortal folly and supernatural tragedy. You serve as the Game Master for a World of Darkness tabletop RPG session played by a single player. Your voice is atmospheric, gothic, and literary — equal parts Poe, Anne Rice, and Neil Gaiman. You speak in rich, evocative prose but never lose clarity of scene.

CORE BEHAVIORAL RULES:
1. TONE — Maintain the World of Darkness atmosphere at all times: moral ambiguity, personal horror, political intrigue, and the ever-present weight of eternity. The world is dark; hope is precious and costs something.

2. IMMERSION — Never break the fourth wall. Never reference "the game," "the rules," "dice pools," or "mechanics" directly to the player. If a mechanical outcome is relevant, describe its narrative effect only (e.g., "Your grip fails you at the crucial moment" rather than "You botched the roll").

3. LORE ACCURACY — You may only assert canonical facts about powers, factions, history, and lore that are present in the retrieved context chunks provided in the user message. If something is not in the retrieved lore, describe it atmospherically without inventing canonical details. Signal uncertainty with phrases like "the elders whisper that..." or "rumors persist..."

4. SCENE CLOSURE — Every scene narration MUST end with exactly three (3) concrete player choices, presented as a numbered list below the narrative prose, preceded by the marker [CHOICES]:
   Example:
   [CHOICES]
   1. [ACTION OPTION A]
   2. [ACTION OPTION B]
   3. [ACTION OPTION C]
   Choices should represent meaningfully different approaches (social, physical, investigative, escape).

5. CHARACTER STATS — When the player attempts actions that would be affected by their character's attributes, abilities, or supernatural powers (from the character sheet provided), factor these into your narrative outcome descriptions. A character with Strength 4 succeeds at feats of power; one with Strength 1 struggles. Do not narrate impossible successes.

6. TENSION TRACKING — Maintain awareness of the session tension level (1-10) provided in the context. At tension 1-3: atmospheric, exploratory pacing. At 4-6: rising dread and complications. At 7-9: desperate urgency in prose. At 10: cinematic, breathless crisis narration.

7. DOOM CLOCK — Reference the doom clock value (0-12) in your pacing. As doom advances, the consequences of inaction become more visible in the world around the character.

8. DOWNTIME — If the scene type is DOWNTIME, shift to introspective, journal-entry tone. Prompt character reflection, relationship development, and personal goal pursuit.

9. LENGTH — Scene narrations should be 150-350 words of prose. NPC dialogue should feel natural and distinct. Never pad; every sentence serves the scene.

10. FORMATTING — Narration prose first. NPC dialogue in quotes. [CHOICES] block at the end. Never use markdown headers or bullet points within the narration prose itself.

System Prompt — Multi-Player Coordinator
text — System Prompt: Multi-Player Coordinator
You are the Storyteller for a group World of Darkness session with multiple players. Your voice remains atmospheric and gothic, but you now coordinate a chorus of characters rather than a solo narrative.

MULTI-PLAYER BEHAVIORAL RULES:
1. CHARACTER ADDRESSING — When narrating an outcome or directing attention to a specific character, use the marker [CHARACTER NAME]: before addressing them. This signals to the Discord bot which player is being directly addressed.

2. SPOTLIGHT MANAGEMENT — Distribute narrative focus equitably. If one character has been prominent in the last 2-3 exchanges, actively create hooks that draw other characters into the scene. No character should be a passive observer for more than 2 consecutive beats.

3. TURN ORDER — In combat or highly structured social conflict scenes, maintain a clear initiative or turn order as provided in the context. Narrate results for the active character, then clearly signal whose turn is next with: [NEXT: CHARACTER_NAME]

4. INTER-PC DYNAMICS — Acknowledge and amplify the interpersonal drama between player characters. NPC reactions to PC group dynamics enrich the scene. If PCs are divided, let the world notice.

5. GROUP CHOICES — End scenes with choices that may affect different characters differently, or require group consensus. Format:
   [GROUP CHOICES]
   1. [OPTION A — particularly relevant for CHARACTER_NAME]
   2. [OPTION B — the group decides together]
   3. [OPTION C — any character can act independently]

6. All other Solo rules (tone, lore accuracy, immersion, tension) apply equally here.

System Prompt — GM Co-Pilot
text — System Prompt: GM Co-Pilot (Assistant GM Mode)
You are an experienced co-Storyteller whispering suggestions to a human Game Master running a World of Darkness session. You are NOT narrating to players. You are a creative partner, a lore sage, and a tactical advisor — speaking only to the GM.

YOUR OUTPUT FORMAT — Always use bracket annotations:
  [SUGGESTION:] — A narrative or scene direction idea
  [RULE:]       — A mechanical rules clarification (cite source book when possible)
  [HOOK:]       — A plot hook, foreshadowing seed, or complication to introduce
  [PACING:]     — Advice on session pacing, tension management, or spotlight balance
  [NPC:]        — An NPC motivation, hidden agenda, or reaction suggestion
  [LORE:]       — A canonical lore fact relevant to the scene (from retrieved context)

BEHAVIORAL RULES:
1. NEVER write narration directed at players. Your output goes to the GM only.
2. Be concise but substantive. A suggestion without reasoning is unhelpful.
3. Offer 2-3 options where possible — the GM chooses, you suggest.
4. Cite source material when providing rules or lore: (V20, p.142) or (WtA20, p.88).
5. Respect the GM's established chronicle. Work with what they've described; do not contradict their established fiction without noting the conflict.
6. When a player action breaks or bends canonical rules, flag it gently under [RULE:].
7. Maintain the gothic, politically complex World of Darkness sensibility in all suggestions — consequences are rarely simple, victories are rarely clean.

python — app/core/prompt_builder.py
"""PromptBuilder — assembles context-rich prompts for the Storyteller Engine. Called before every LLM request to construct system + user message pairs."""
from __future__ import annotations

from typing import Any, Dict, List, Optional

from app.db.models import Character, GameSession, NPC
from app.enums import SessionMode
from app.core.templates import (
    SCENE_NARRATION_TEMPLATE,
    COMBAT_NARRATION_TEMPLATE,
    DOWNTIME_REFLECTION_TEMPLATE,
    MULTI_CHARACTER_SCENE_TEMPLATE,
    GM_SCENE_SUGGESTION_TEMPLATE,
    NPC_VOICE_TEMPLATE,
    ENCOUNTER_GENERATOR_TEMPLATE,
    LORE_LOOKUP_TEMPLATE,
    SOLO_SYSTEM_PROMPT,
    GROUP_SYSTEM_PROMPT,
    ASSISTANT_GM_SYSTEM_PROMPT,
)


class PromptBuilder:
    """
    Assembles complete prompt objects (system + user) for all Storyteller AI modes.
    Accepts pre-fetched lore chunks, session state, and character data.
    """

    # ?? System Prompt Selection ????????????????????????????????????????????
    def build_system_prompt(self, mode: SessionMode, session: GameSession) -> str:
        """
        Returns the appropriate system prompt string for the given session mode.
        Injects session-level customizations (chronicle name, setting splat).
        """
        base_prompts = {
            SessionMode.SOLO: SOLO_SYSTEM_PROMPT,
            SessionMode.GROUP: GROUP_SYSTEM_PROMPT,
            SessionMode.ASSISTANT_GM: ASSISTANT_GM_SYSTEM_PROMPT,
        }
        base = base_prompts.get(mode, SOLO_SYSTEM_PROMPT)

        # Inject chronicle name as context header
        header = f"[CHRONICLE: {session.name} | SETTING: {session.setting.value.upper()}]\n\n"
        return header + base

    # ?? Narration Prompt ??????????????????????????????????????????????????
    def build_narration_prompt(
        self,
        session: GameSession,
        player_input: str,
        lore_chunks: List[str],
        acting_character: Optional[Character] = None,
        combat_mode: bool = False,
        downtime_mode: bool = False,
    ) -> str:
        """
        Assembles the user-turn prompt for narration.
        Selects template (scene / combat / downtime) based on session state flags.
        """
        state = session.state_json or {}
        lore_text = self._format_lore_chunks(lore_chunks)
        npc_text = self._format_npcs(state.get("active_npcs", []))
        recent_events = self._format_recent_events(state.get("scene_history", [])[-5:])

        if downtime_mode:
            template = DOWNTIME_REFLECTION_TEMPLATE
        elif combat_mode:
            template = COMBAT_NARRATION_TEMPLATE
        elif session.mode == SessionMode.GROUP:
            template = MULTI_CHARACTER_SCENE_TEMPLATE
        else:
            template = SCENE_NARRATION_TEMPLATE

        char = acting_character
        return template.format(
            session_name=session.name,
            setting=session.setting.value,
            scene_title=state.get("scene_title", "Unknown Scene"),
            location=state.get("location", "Unknown Location"),
            time_of_night=state.get("time_of_night", "Unknown Hour"),
            tension_level=state.get("tension_level", 5),
            doom_clock=state.get("doom_clock", 0),
            state_machine_state=state.get("machine_state", "scene_open"),
            character_name=char.name if char else "Unknown",
            player_name=char.player_name if char else "Unknown",
            splat=char.splat.value if char else "unknown",
            clan_tribe_tradition=char.clan_tribe_tradition if char else "",
            generation_rank_arete=char.generation_rank_arete if char else 5,
            attributes_summary=self._summarize_attributes(char),
            key_abilities=self._summarize_abilities(char),
            powers_list=self._summarize_powers(char),
            health_summary=self._summarize_health(char),
            pool_current=char.blood_pool_rage_quintessence if char else 0,
            pool_max=10,
            willpower_current=char.willpower if char else 5,
            willpower_max=10,
            active_conditions=state.get("active_conditions", "None"),
            npc_list=npc_text,
            recent_events=recent_events,
            lore_chunks=lore_text,
            player_input=player_input,
            # Group mode extras
            player_count=len(state.get("player_characters", [])),
            character_block=self._format_character_block(session),
            acting_character_name=char.name if char else "Player",
            acting_player_name=char.player_name if char else "Player",
        )

    # ?? NPC Voice Prompt ??????????????????????????????????????????????????
    def build_npc_voice_prompt(self, npc: NPC, context: str) -> str:
        """Assembles the user-turn prompt for NPC dialogue generation."""
        vp = npc.voice_profile_json or {}
        return NPC_VOICE_TEMPLATE.format(
            npc_name=npc.name,
            splat=npc.splat.value if npc.splat else "unknown",
            clan_tribe=npc.clan_tribe or "Unknown",
            archetype=npc.archetype or "Unknown",
            disposition=npc.disposition.value,
            current_goal="[Not specified — infer from secrets]",
            voice_tone=vp.get("tone", "measured"),
            speech_patterns=vp.get("speech_patterns", "formal"),
            vocabulary_level=vp.get("vocabulary_level", "educated"),
            accent_notes=vp.get("accent_notes", "none"),
            mannerisms=vp.get("mannerisms", "none noted"),
            backstory_excerpt=str(npc.secrets_json)[:300],
            scene_context=context,
            gm_request="Generate dialogue options for this NPC in the current scene.",
        )

    # ?? Encounter Prompt ??????????????????????????????????????????????????
    def build_encounter_prompt(
        self,
        location: str,
        mood: str,
        threat_level: int,
        active_factions: List[str],
        session: Optional[GameSession] = None,
        lore_chunks: Optional[List[str]] = None,
    ) -> str:
        """Assembles the encounter generation prompt."""
        threat_labels = {1: "Trivial", 2: "Minor", 3: "Moderate", 4: "Severe", 5: "Catastrophic"}
        state = session.state_json if session else {}
        return ENCOUNTER_GENERATOR_TEMPLATE.format(
            location=location,
            time_of_night=state.get("time_of_night", "witching hour"),
            mood=mood,
            threat_level=threat_level,
            threat_label=threat_labels.get(threat_level, "Unknown"),
            active_factions=", ".join(active_factions) if active_factions else "None specified",
            pc_summary=self._format_pc_summary(session),
            tension=state.get("tension_level", 5),
            doom=state.get("doom_clock", 0),
            lore_chunks=self._format_lore_chunks(lore_chunks or []),
        )

    # ?? Private Helpers ???????????????????????????????????????????????????
    def _format_lore_chunks(self, chunks: List[str]) -> str:
        if not chunks:
            return "[No lore retrieved — use atmospheric description only]"
        return "\n---\n".join(f"[Chunk {i+1}]: {chunk}" for i, chunk in enumerate(chunks))

    def _format_npcs(self, active_npcs: List[Dict]) -> str:
        if not active_npcs:
            return "None active in scene."
        lines = []
        for npc in active_npcs:
            lines.append(
                f"  {npc.get('name', 'Unknown')} — {npc.get('disposition', 'neutral')} | "
                f"Goal: {npc.get('current_goal', 'unknown')}"
            )
        return "\n".join(lines)

    def _format_recent_events(self, events: List[Dict]) -> str:
        if not events:
            return "No prior events recorded."
        lines = []
        for evt in events:
            lines.append(f"  [{evt.get('event_type', 'event')}] {evt.get('summary', '')}")
        return "\n".join(lines)

    def _summarize_attributes(self, char: Optional[Character]) -> str:
        if not char or not char.attributes_json:
            return "Unknown"
        attrs = char.attributes_json
        return (
            f"STR {attrs.get('strength', '?')} DEX {attrs.get('dexterity', '?')} "
            f"STA {attrs.get('stamina', '?')} | "
            f"CHA {attrs.get('charisma', '?')} MAN {attrs.get('manipulation', '?')} "
            f"APP {attrs.get('appearance', '?')} | "
            f"PER {attrs.get('perception', '?')} INT {attrs.get('intelligence', '?')} "
            f"WIT {attrs.get('wits', '?')}"
        )

    def _summarize_abilities(self, char: Optional[Character]) -> str:
        if not char or not char.abilities_json:
            return "Unknown"
        ab = char.abilities_json
        relevant = {k: v for k, v in ab.items() if isinstance(v, int) and v >= 2}
        return ", ".join(f"{k.capitalize()} {v}" for k, v in list(relevant.items())[:8])

    def _summarize_powers(self, char: Optional[Character]) -> str:
        if not char or not char.disciplines_gifts_spheres_json:
            return "None"
        powers = char.disciplines_gifts_spheres_json
        return ", ".join(f"{k} {v}" for k, v in list(powers.items())[:6])

    def _summarize_health(self, char: Optional[Character]) -> str:
        if not char or not char.health_levels_json:
            return "Unknown"
        hl = char.health_levels_json
        return f"Current: {hl.get('current_level', 'Healthy')} | Aggravated: {hl.get('aggravated', 0)}"

    def _format_character_block(self, session: Optional[GameSession]) -> str:
        if not session or not session.characters:
            return "No characters registered."
        lines = []
        for char in session.characters:
            lines.append(
                f"Name: {char.name} | Player: {char.player_name} | "
                f"Splat: {char.splat.value} | Health: {self._summarize_health(char)}\n"

"""PDF Parser for White Wolf sourcebooks. Uses PyMuPDF (fitz) for text extraction with font metadata. Tailored to handle common OCR artifacts and layout patterns in older White Wolf PDFs and modern Onyx Path releases."""
from __future__ import annotations

import re
import unicodedata
from pathlib import Path
from typing import Optional

import fitz  # PyMuPDF

# ?? Constants ????????????????????????????????????????????????????????????????
TEXT_DENSITY_THRESHOLD = 0.05
HEADER_FONT_SIZE_THRESHOLD = 13.0

PAGE_HEADER_FOOTER_PATTERNS = [
    re.compile(r"^\d{1,3}$"),
    re.compile(r"^(Chapter|CHAPTER)\s+\d+"),
    re.compile(r"^(White Wolf|Onyx Path|Paradox)", re.IGNORECASE),
    re.compile(r"World of Darkness", re.IGNORECASE),
]

LIGATURE_MAP = {
    "\ufb01": "fi",
    "\ufb02": "fl",
    "\ufb00": "ff",
    "\ufb03": "ffi",
    "\ufb04": "ffl",
    "\u2019": "'",
    "\u2018": "'",
    "\u201c": '"',
    "\u201d": '"',
    "\u2013": "-",
    "\u2014": "--",
    "\u2026": "...",
}


class WhiteWolfPDFParser:
    """Extracts structured text from White Wolf/Onyx Path sourcebook PDFs."""

    def __init__(self, pdf_path: str | Path, source_book_name: str):
        self.pdf_path = Path(pdf_path)
        self.source_book_name = source_book_name
        self._doc: Optional[fitz.Document] = None

    def __enter__(self) -> "WhiteWolfPDFParser":
        self._doc = fitz.open(str(self.pdf_path))
        return self

    def __exit__(self, *args) -> None:
        if self._doc:
            self._doc.close()

    def parse(self) -> list[dict]:
        """Main entry point returning structured text segments."""
        if not self._doc:
            raise RuntimeError("Parser not opened. Use as context manager.")

        results = []
        current_chapter = None
        current_section = None

        for page_num in range(len(self._doc)):
            page = self._doc[page_num]

            if self._is_artwork_heavy(page):
                continue

            blocks = page.get_text("dict", flags=fitz.TEXT_PRESERVE_LIGATURES)["blocks"]

            for block in blocks:
                if block.get("type") != 0:
                    continue

                for line in block.get("lines", []):
                    for span in line.get("spans", []):
                        raw_text = span.get("text", "").strip()
                        if not raw_text:
                            continue

                        cleaned = self._clean_text(raw_text)
                        if not cleaned:
                            continue

                        font_size = span.get("size", 11.0)
                        font_flags = span.get("flags", 0)
                        is_bold = bool(font_flags & 2**4)
                        is_header = (
                            font_size >= HEADER_FONT_SIZE_THRESHOLD and is_bold and len(cleaned) < 120
                        )

                        if is_header:
                            if font_size >= 18.0:
                                current_chapter = cleaned
                                current_section = None
                            else:
                                current_section = cleaned

                        results.append({
                            "page": page_num + 1,
                            "chapter": current_chapter,
                            "section": current_section,
                            "text": cleaned,
                            "source_book": self.source_book_name,
                            "is_header": is_header,
                            "font_size": round(font_size, 1),
                        })

        return self._merge_into_paragraphs(results)

    def _is_artwork_heavy(self, page: fitz.Page) -> bool:
        text = page.get_text("text")
        if not text:
            return True
        rect = page.rect
        page_area = rect.width * rect.height
        density = len(text) / max(page_area, 1)
        return density < TEXT_DENSITY_THRESHOLD

    def _clean_text(self, text: str) -> str:
        for ligature, replacement in LIGATURE_MAP.items():
            text = text.replace(ligature, replacement)

        text = unicodedata.normalize("NFKC", text)

        stripped = text.strip()
        for pattern in PAGE_HEADER_FOOTER_PATTERNS:
            if pattern.match(stripped):
                return ""

        text = re.sub(r"\s+", " ", text).strip()

        if re.fullmatch(r"[•??\-–—_]+", text):
            return ""

        return text

    def _merge_into_paragraphs(self, segments: list[dict]) -> list[dict]:
        if not segments:
            return []

        merged = []
        buffer = []
        current_meta = segments[0].copy()

        for seg in segments:
            if seg["is_header"]:
                if buffer:
                    combined = current_meta.copy()
                    combined["text"] = " ".join(buffer)
                    combined["is_header"] = False
                    merged.append(combined)
                    buffer = []
                merged.append(seg)
                current_meta = seg.copy()
            else:
                if seg["page"] == current_meta["page"] and seg["section"] == current_meta["section"]:
                    buffer.append(seg["text"])
                else:
                    if buffer:
                        combined = current_meta.copy()
                        combined["text"] = " ".join(buffer)
                        combined["is_header"] = False
                        merged.append(combined)
                        buffer = []
                    buffer = [seg["text"]]
                    current_meta = seg.copy()

        if buffer:
            combined = current_meta.copy()
            combined["text"] = " ".join(buffer)
            combined["is_header"] = False
            merged.append(combined)

        return merged


def parse_pdf(pdf_path: str, source_book_name: str) -> list[dict]:
    """Convenience function wrapping the parser context manager."""
    with WhiteWolfPDFParser(pdf_path, source_book_name) as parser:
        return parser.parse()

def classify_chunk(text: str) -> LoreCategory:
    if RITUAL_PATTERN.search(text):
        return LoreCategory.RITUAL
    if POWER_PATTERN.search(text) and RULE_PATTERN.search(text):
        return LoreCategory.POWER
    if RULE_PATTERN.search(text):
        return LoreCategory.RULE
    if NPC_TEMPLATE_PATTERN.search(text):
        return LoreCategory.NPC_TEMPLATE
    if LOCATION_PATTERN.search(text):
        return LoreCategory.LOCATION
    return LoreCategory.LORE


def detect_splat_relevance(text: str, chapter: Optional[str]) -> list[str]:
    combined = f"{text} {chapter or ''}".lower()
    found = []
    splat_keywords = {
        Splat.VAMPIRE: ["kindred", "vampire", "embrace", "blood pool", "torpor", "clan"],
        Splat.WEREWOLF: ["garou", "werewolf", "rage", "gnosis", "caern", "tribe", "gift"],
        Splat.MAGE: ["mage", "awakened", "sphere", "quintessence", "paradox", "tradition"],
        Splat.CHANGELING: ["changeling", "fae", "glamour", "dreaming", "seeming", "kith"],
        Splat.HUNTER: ["hunter", "edge", "creed", "imbued"],
        Splat.WRAITH: ["wraith", "shadow", "corpus", "pathos", "haunt", "arcanoi"],
    }
    for splat, keywords in splat_keywords.items():
        if any(kw in combined for kw in keywords):
            found.append(splat.value)
    return found if found else ["universal"]


class ChunkEmbedder:
    """Handles text chunking, embedding generation, and storage."""

    def __init__(self, source_book: str, splat: Optional[str] = None):
        self.source_book = source_book
        self.splat = splat
        self.splitter = RecursiveCharacterTextSplitter(
            chunk_size=800,
            chunk_overlap=150,
            separators=["\n\n", "\n", ". ", "! ", "? ", " ", ""],
        )
        self.openai_client = OpenAI(api_key=settings.OPENAI_API_KEY)
        self.chroma_client = get_chroma_client()
        self.collection = self.chroma_client.get_or_create_collection(
            name=settings.CHROMA_COLLECTION_NAME,
            metadata={"hnsw:space": "cosine"},
        )

    def embed_segments(self, segments: list[dict]) -> list[dict]:
        body_segments = [s for s in segments if not s.get("is_header")]
        full_texts = [s["text"] for s in body_segments]
        combined = "\n\n".join(full_texts)

        raw_chunks = self.splitter.split_text(combined)
        lore_entries = []

        for i, chunk_text in enumerate(raw_chunks):
            if len(chunk_text.strip()) < 50:
                continue

            category = classify_chunk(chunk_text)
            splat_relevance = detect_splat_relevance(
                chunk_text,
                segments[min(i, len(segments) - 1)].get("chapter"),
            )

            embedding_response = self.openai_client.embeddings.create(
                input=chunk_text,
                model=settings.OPENAI_EMBEDDING_MODEL,
            )
            embedding = embedding_response.data[0].embedding
            chunk_id = str(uuid.uuid4())

            ref_seg = segments[min(i * max(len(segments) // max(len(raw_chunks), 1), 1), len(segments) - 1)]

            metadata = {
                "source_book": self.source_book,
                "page": ref_seg.get("page", 0),
                "chapter": ref_seg.get("chapter") or "",
                "section": ref_seg.get("section") or "",
                "category": category.value,
                "splat_relevance": ",".join(splat_relevance),
                "chunk_index": i,
            }

            self.collection.add(
                ids=[chunk_id],
                embeddings=[embedding],
                documents=[chunk_text],
                metadatas=[metadata],
            )

            lore_entries.append({
                "id": chunk_id,
                "title": f"{self.source_book} — {ref_seg.get('section') or ref_seg.get('chapter') or 'Passage'}",
                "source_book": self.source_book,
                "page_number": ref_seg.get("page"),
                "category": category,
                "content_text": chunk_text,
                "embedding_id": chunk_id,
                "tags_json": splat_relevance,
                "splat_relevance_json": splat_relevance,
            })

        return lore_entries

        self.update_state(
            state="PROGRESS",

                "total": total,

@celery_app.task(
    bind=True,
    base=IngestionTask,
    name="tasks.ingest_pdf",
    queue="ingestion",
    max_retries=3,
    default_retry_delay=30,
    acks_late=True,
)
def ingest_pdf_task(
    self: IngestionTask,
    file_path: str,
    source_book_name: str,
    splat: str,
) -> dict:
    """Full PDF ingestion pipeline as a single Celery task."""
    logger.info(f"Beginning ingestion: {source_book_name}")

    self.update_progress(0, 100, f"Parsing PDF: {source_book_name}")
    try:
        segments = parse_pdf(file_path, source_book_name)
        logger.info(f"Extracted {len(segments)} text segments from {source_book_name}")
    except Exception as exc:
        logger.error(f"PDF parsing failed for {file_path}: {exc}")
        raise self.retry(exc=exc)

    self.update_progress(25, 100, "Chunking and embedding text...")
    try:
        embedder = ChunkEmbedder(source_book=source_book_name, splat=splat)
        lore_entries = embedder.embed_segments(segments)
        logger.info(f"Created {len(lore_entries)} embedded chunks")
    except Exception as exc:
        logger.error(f"Embedding failed for {source_book_name}: {exc}")
        raise self.retry(exc=exc)

    self.update_progress(75, 100, "Writing LoreEntry records to database...")
    try:
        lore_entries_created = _write_lore_entries_sync(lore_entries)
    except Exception as exc:
        logger.error(f"Database write failed: {exc}")
        raise self.retry(exc=exc)

    self.update_progress(95, 100, "Cleaning up temp files...")
    try:
        Path(file_path).unlink(missing_ok=True)
    except Exception:
        pass

    self.update_progress(100, 100, "Ingestion complete.")
    logger.info(f"Ingestion complete: {source_book_name} — {len(lore_entries)} chunks")

    return {
        "status": "complete",
        "source_book": source_book_name,
        "segments_parsed": len(segments),
        "chunks_created": len(lore_entries),
        "lore_entries_created": lore_entries_created,
    }


def _write_lore_entries_sync(lore_entries: list[dict]) -> int:
    """Synchronous wrapper for async DB write used within Celery worker context."""
    from app.db.session import AsyncSessionLocal
    from app.db.models import LoreEntry

    async def _write():
        written = 0
        async with AsyncSessionLocal() as session:
            for entry_data in lore_entries:
                entry = LoreEntry(
                    id=entry_data["id"],
                    title=entry_data["title"],
                    source_book=entry_data["source_book"],
                    page_number=entry_data.get("page_number"),
                    category=entry_data["category"],
                    content_text=entry_data["content_text"],
                    embedding_id=entry_data["embedding_id"],
                    tags_json=entry_data["tags_json"],
                    splat_relevance_json=entry_data["splat_relevance_json"],
                )
                session.add(entry)
                written += 1
            await session.commit()
        return written

    return asyncio.run(_write())

Section 6 — State Engine

6.1 Overview

The narrative state engine is the beating heart beneath the Storyteller AI's skin — the machinery that ensures the AI does not simply respond to each player message in isolation, but maintains coherent, evolving awareness of the chronicle's dramatic arc. Without persistent state, the AI is an amnesiac narrator. With it, the AI is a living chronicle.
The state engine tracks: the current scene and its location, all active NPCs and their shifting dispositions, each player character's current physical and supernatural condition, a tension meter that modulates the prose register, a doom clock that creates urgency and consequence, a momentum tracker that reflects who is winning the narrative struggle, and a rolling history of the last ten scene events for context injection.
State is persisted to the state_json column of the GameSession table after every narrative exchange, ensuring full recovery from crashes, server restarts, and Discord disconnections.

import logging
from typing import Dict

from app.enums import StateMachineState
from app.core.state_machine_models import SessionState

logger = logging.getLogger("storyteller.state")

VALID_TRANSITIONS: Dict[StateMachineState, list[StateMachineState]] = {
    StateMachineState.SCENE_OPEN: [
        StateMachineState.PLAYER_ACTING,
        StateMachineState.DOWNTIME,
        StateMachineState.SESSION_END,
    ],
    StateMachineState.PLAYER_ACTING: [
        StateMachineState.NPC_REACTING,
        StateMachineState.COMBAT,
        StateMachineState.SOCIAL_CONFLICT,
        StateMachineState.SCENE_TRANSITION,
    ],
    StateMachineState.NPC_REACTING: [
        StateMachineState.PLAYER_ACTING,
        StateMachineState.COMBAT,
        StateMachineState.SOCIAL_CONFLICT,
        StateMachineState.SCENE_OPEN,
    ],
    StateMachineState.COMBAT: [
        StateMachineState.PLAYER_ACTING,
        StateMachineState.SCENE_TRANSITION,
        StateMachineState.DOWNTIME,
        StateMachineState.SESSION_END,
    ],
    StateMachineState.SOCIAL_CONFLICT: [
        StateMachineState.PLAYER_ACTING,
        StateMachineState.NPC_REACTING,
        StateMachineState.SCENE_TRANSITION,
    ],
    StateMachineState.DOWNTIME: [
        StateMachineState.SCENE_OPEN,
        StateMachineState.SESSION_END,
    ],
    StateMachineState.SCENE_TRANSITION: [StateMachineState.SCENE_OPEN],
    StateMachineState.SESSION_END: [],
}


class StorytellerStateMachine:
    """
    Manages the narrative state of a World of Darkness session.
    Handles transitions, meter updates, persistence, and doom events.
    """

    def __init__(self, state: SessionState):
        self.state = state

    # ?? State Transitions ????????????????????????????????????????????????
    def transition_to(self, new_state: StateMachineState, reason: str = "") -> bool:
        """
        Attempts a state transition. Validates against allowed transition table.

        Returns True if successful, False if the transition is not valid.
        """
        current = self.state.machine_state
        if new_state not in VALID_TRANSITIONS.get(current, []):
            logger.warning(
                f"Invalid transition: {current.value} -> {new_state.value} | Reason: {reason}"
            )
            return False

        logger.info(f"State transition: {current.value} -> {new_state.value} | {reason}")
        self.state.machine_state = new_state

        # Reset combat round on combat entry
        if new_state == StateMachineState.COMBAT and self.state.combat_round == 0:
            self.state.combat_round = 1

        # Reset combat round on combat exit
        if current == StateMachineState.COMBAT and new_state != StateMachineState.COMBAT:
            self.state.combat_round = 0
            self.state.initiative_order = []

        return True

    # ?? Guard Conditions ?????????????????????????????????????????????????
    def can_enter_combat(self, player_declared_attack: bool = False) -> bool:
        """Combat entry requires tension >= 6 OR explicit player attack declaration."""
        return self.state.tension_level >= 6 or player_declared_attack

    def can_enter_downtime(self) -> bool:
        """Downtime only available when not in active combat or crisis."""
        return self.state.machine_state not in [
            StateMachineState.COMBAT,
            StateMachineState.SOCIAL_CONFLICT,
        ]

    # ?? Narrative Meter Updates ??????????????????????????????????????????
    def update_tension(self, delta: int) -> int:
        """
        Adjusts tension level by delta. Clamps to [1, 10].
        If tension has been at 10 for 2+ consecutive scenes, tracks for doom advancement.

        Returns the new tension level.
        """
        old_tension = self.state.tension_level
        self.state.tension_level = max(1, min(10, self.state.tension_level + delta))

        if self.state.tension_level == 10:
            self.state.consecutive_scenes_at_max_tension += 1
        else:
            self.state.consecutive_scenes_at_max_tension = 0

        logger.debug(f"Tension: {old_tension} -> {self.state.tension_level} (?{delta:+d})")
        return self.state.tension_level

    def update_momentum(self, delta: int) -> int:
        """Adjusts momentum by delta. Clamps to [-5, +5]."""
        self.state.momentum = max(-5, min(5, self.state.momentum + delta))
        return self.state.momentum

    def advance_doom(self, n: int = 1) -> int:
        """
        Advances the doom clock by n steps. Clamps to [0, 12].
        If doom reaches 12, triggers the catastrophic doom event.

        Returns the new doom value.
        """
        old_doom = self.state.doom_clock
        self.state.doom_clock = min(12, self.state.doom_clock + n)
        logger.warning(f"Doom Clock: {old_doom} -> {self.state.doom_clock}")

        if self.state.doom_clock >= 12:
            self.trigger_doom_event()

        return self.state.doom_clock

    def trigger_doom_event(self) -> dict:
        """
        Called when the doom clock reaches 12.
        Generates a catastrophic narrative event descriptor and resets the clock to 8
        (allowing the chronicle to continue, scarred, rather than abruptly ending).

        Returns a dict describing the catastrophic event for injection into the next prompt.
        """
        logger.critical("??  DOOM CLOCK AT 12 — TRIGGERING CATASTROPHIC EVENT")

        doom_event = {
            "type": "doom_event",
            "severity": "catastrophic",
            "description": (
                "The accumulated weight of inaction and consequence crashes down. "
                "Something irreversible has occurred. An ally is lost, a haven destroyed, "
                "a secret revealed to the worst possible audience, or a power awakens "
                "that was better left sleeping. The chronicle will never be the same."
            ),
            "narrative_instruction": (
                "DOOM EVENT TRIGGERED — The Storyteller must narrate a catastrophic, "
                "chronicle-altering event that reflects the accumulated failures and threats "
                "of this session. This is not the end — it is the darkest hour before dawn "
                "costs everything. Convey weight, loss, and terrible consequence."
            ),
        }

        # Reset doom to 8 — chronicle continues but scarred
        self.state.doom_clock = 8
        self.state.tension_level = 10
        self.state.consecutive_scenes_at_max_tension = 0

        return doom_event

    # ?? Player Action Processing ?????????????????????????????????????????
    def process_player_action(self, action_text: str) -> dict:
        """
        Analyzes player action text and returns a recommended state delta.
        Detects attack declarations, social engagement, stealth, retreat.

        Returns a dict of recommended state changes to apply before LLM call.
        """
        action_lower = action_text.lower()
        delta: dict = {
            "tension_delta": 0,
            "momentum_delta": 0,
            "suggested_transition": None,
        }

        # Attack detection ? suggest COMBAT transition
        attack_keywords = ["attack", "strike", "shoot", "bite", "slash", "fight", "kill", "draw weapon"]
        if any(kw in action_lower for kw in attack_keywords):
            delta["suggested_transition"] = StateMachineState.COMBAT
            delta["tension_delta"] = 2

        # Social manipulation ? suggest SOCIAL_CONFLICT
        social_keywords = ["intimidate", "persuade", "seduce", "command", "dominate", "convince", "threaten"]
        if any(kw in action_lower for kw in social_keywords):
            if self.state.machine_state != StateMachineState.COMBAT:
                delta["suggested_transition"] = StateMachineState.SOCIAL_CONFLICT
                delta["tension_delta"] = 1

        # Retreat / flee ? reduce tension
        retreat_keywords = ["flee", "escape", "run", "retreat", "hide", "go to ground"]
        if any(kw in action_lower for kw in retreat_keywords):
            delta["tension_delta"] = -1
            delta["momentum_delta"] = -1

        return delta

    # ?? Auto-Advance Doom Logic ??????????????????????????????????????????
    def tick_scene(self) -> None:
        """
        Called at the end of each scene narration turn.
        Applies automatic doom advancement triggers.
        """
        # Doom advances if tension has been at 10 for 2+ consecutive scenes
        if self.state.consecutive_scenes_at_max_tension >= 2:
            self.advance_doom(1)
            logger.warning("Doom advanced: sustained maximum tension.")

        # Doom advances if momentum deeply negative (players losing)
        if self.state.momentum <= -4:
            self.advance_doom(1)
            logger.warning("Doom advanced: severe negative momentum.")

    # ?? Serialization ????????????????????????????????????????????????????
    def serialize(self) -> dict:
        """Serialize the current state to a dict for storage in session.state_json."""
        return self.state.model_dump(mode="json")

    @classmethod
    def deserialize(cls, state_dict: dict) -> "StorytellerStateMachine":
        """Reconstruct a StorytellerStateMachine from a stored state_json dict."""
        state = SessionState.model_validate(state_dict)
        return cls(state=state)
6.4 Context Window Management

Every narration call must assemble a prompt that fits within the model's context window. The Storyteller AI targets a conservative 12,000-token budget per request, leaving headroom for the model's response and guardrailing against unexpected input size spikes.

"""Context assembly for prompt construction — token-aware."""
import tiktoken
from typing import List

from app.core.state_machine_models import SessionState

TIKTOKEN_ENCODER = tiktoken.get_encoding("cl100k_base")  # GPT-4o encoding
TOKEN_BUDGET = 12_000
SYSTEM_PROMPT_RESERVE = 1_500    # Tokens reserved for system prompt
RESPONSE_RESERVE = 2_048         # Tokens reserved for model response output
AVAILABLE_BUDGET = TOKEN_BUDGET - SYSTEM_PROMPT_RESERVE - RESPONSE_RESERVE
# = 8,452 tokens for dynamic context


def count_tokens(text: str) -> int:
    """Returns the token count for a string using GPT-4o tokenizer."""
    return len(TIKTOKEN_ENCODER.encode(text))


def context_assembler(
    state: SessionState,
    lore_chunks: List[str],
    player_input: str,
    character_summary: str,
    max_tokens: int = AVAILABLE_BUDGET,
) -> dict:
    """
    Assembles context components within the token budget.

    Priority order (highest to lowest):
      1. Player input (always included)
      2. Character summary (always included)
      3. Current state snapshot (always included)
      4. Active NPC list (always included)
      5. Last 5 scene events (trimmed if needed)
      6. Retrieved lore chunks (top 3, trimmed to fit)

    Returns dict with final assembled text components and token usage stats.
    """
    components = {}
    total_tokens = 0

    # ?? Always-include components ??????????????????????????????????????
    pi_tokens = count_tokens(player_input)
    char_tokens = count_tokens(character_summary)
    state_snap = _build_state_snapshot(state)
    state_tokens = count_tokens(state_snap)
    npc_text = _build_npc_list(state)
    npc_tokens = count_tokens(npc_text)

    core_tokens = pi_tokens + char_tokens + state_tokens + npc_tokens
    total_tokens += core_tokens

    components["player_input"] = player_input
    components["character_summary"] = character_summary
    components["state_snapshot"] = state_snap
    components["npc_list"] = npc_text

    # ?? Scene history (last 5 events, trim if over budget) ?????????????
    remaining = max_tokens - total_tokens
    history_text = _build_scene_history(state, max_events=5)
    history_tokens = count_tokens(history_text)

    if history_tokens <= remaining:
        components["scene_history"] = history_text
        total_tokens += history_tokens
        remaining -= history_tokens
    else:
        # Include fewer events
        for n in [3, 2, 1]:
            trimmed = _build_scene_history(state, max_events=n)
            t = count_tokens(trimmed)
            if t <= remaining:
                components["scene_history"] = trimmed
                total_tokens += t
                remaining -= t
                break
        else:
            components["scene_history"] = "[Scene history omitted — token budget]"

    # ?? Lore chunks (top 3, fit within remaining) ??????????????????????
    lore_included = []
    for chunk in lore_chunks[:3]:
        chunk_tokens = count_tokens(chunk)
        if chunk_tokens <= remaining:
            lore_included.append(chunk)
            remaining -= chunk_tokens
        else:
            break  # No more room

    components["lore_chunks"] = lore_included
    components["token_usage"] = {
        "total": total_tokens + sum(count_tokens(c) for c in lore_included),
        "budget": max_tokens,
        "lore_chunks_included": len(lore_included),
    }

    return components


def _build_state_snapshot(state: SessionState) -> str:
    return (
        f"Scene: {state.scene_title} | Location: {state.location} | "
        f"Time: {state.time_of_night} | Mode: {state.machine_state.value} | "
        f"Tension: {state.tension_level}/10 | Doom: {state.doom_clock}/12 | "
        f"Momentum: {state.momentum:+d}"
    )


def _build_npc_list(state: SessionState) -> str:
    if not state.active_npcs:
        return "No NPCs active in scene."
    lines = [
        f"  {npc.name} — {npc.disposition.value} | Goal: {npc.current_goal}"
        for npc in state.active_npcs if npc.is_present_in_scene
    ]
    return "\n".join(lines) or "No NPCs present."


def _build_scene_history(state: SessionState, max_events: int = 5) -> str:
    events = state.scene_history[-max_events:]
    if not events:
        return "No prior events recorded."
    lines = [f"  [{e.event_type}] {e.summary}" for e in events]
    return "\n".join(lines)
6.5 Tension & Doom System

The tension and doom systems are the Storyteller AI's primary tools for conveying the oppressive weight of the World of Darkness narrative. They are not mere metadata fields — they directly modulate the AI's narration register and trigger automatic consequence events.

Doom Clock (0–12)
The doom clock represents the chronicle's inexorable march toward catastrophe. It advances when antagonists act unopposed, when tension stays at maximum, when players fail at critical moments, or when the GM manually advances it for dramatic purpose. It does not automatically end the chronicle — it marks the approach of an inevitable reckoning.
? Doom 0–3: The shadow falls but has not yet touched the players' world.
? Doom 4–6: The players' enemies grow bolder; NPCs whisper of coming storm.
? Doom 7–9: Consequences of earlier failures begin manifesting visibly.
? Doom 10–11: Catastrophe is close; every scene carries it in the air.
? Doom 12: trigger_doom_event() fires — irreversible chronicle alteration. Clock resets to 8.

? DESIGN TIP — Doom as Story Tool
The doom clock should feel threatening but never arbitrary. Always telegraph what is advancing it through NPC dialogue, environmental details, and scene descriptions. Players who pay attention should be able to anticipate doom events and choose to act (or choose not to, carrying the weight of consequence). The Storyteller AI is instructed to embed these telegraphs automatically as doom rises.

        logger.info("Slash commands synced.")

    async def on_ready(self) -> None:
        logger.info(f"Storyteller Bot awakened as {self.user} (ID: {self.user.id})")
        await self.change_presence(
            activity=discord.Activity(
                type=discord.ActivityType.watching,
                name="the darkness gather...",
            )
        )

    async def on_message(self, message: discord.Message) -> None:
        """
        Routes non-command messages in the designated storyteller channel
        to the narration endpoint as player actions.
        """
        if message.author.bot:
            return

        guild_data = self._guild_sessions.get(message.guild.id if message.guild else 0)
        if not guild_data:
            return  # No active session for this guild

        storyteller_channel_id = guild_data.get("channel_id")
        if message.channel.id != storyteller_channel_id:
            await self.process_commands(message)
            return

        # Rate limit check
        if not await self.rate_limiter.check_cooldown(message.guild.id):
            await message.add_reaction("?")
            return

        if not await self.rate_limiter.check_token_budget(message.guild.id):
            await message.channel.send(
                "This chronicle has exhausted its daily token budget. "
                "The darkness falls silent until tomorrow.",
                delete_after=15,
            )
            return

        # Send to narration endpoint
        async with message.channel.typing():
            try:
                result = await self.api_client.narrate(
                    session_id=guild_data["session_id"],
                    player_input=message.content,
                    character_id=guild_data.get("default_character_id"),
                )
                view = NarrativeResponseView(bot=self, session_id=guild_data["session_id"])
                embed = self._build_narration_embed(result, message.author)
                await message.channel.send(embed=embed, view=view)
                await self.rate_limiter.record_usage(
                    message.guild.id,
                    result.get("token_count", 500),
                )
            except Exception as e:
                logger.error(f"Narration error: {e}", exc_info=True)
                await message.channel.send(
                    "The Storyteller's voice falters. An error has occurred. "
                    "Try again or use /session-status to check session health."
                )

    def register_guild_session(
        self,
        guild_id: int,
        session_id: str,
        channel_id: int,
        gm_user_id: int,
    ) -> None:
        """Registers an active session for a guild."""
        self._guild_sessions[guild_id] = {
            "session_id": session_id,
            "channel_id": channel_id,
            "gm_user_id": gm_user_id,
        }

    def get_guild_session(self, guild_id: int) -> Optional[dict]:
        return self._guild_sessions.get(guild_id)

def is_gm():
    """Check decorator — user must have the configured GM role."""

        settings = get_settings()
        gm_role = discord.utils.get(interaction.guild.roles, name=settings.DISCORD_GM_ROLE_NAME)

7.7 Narrative Channel Flow

The #storyteller-channel pattern is the beating heart of the player-facing experience. Once a session is started and a channel designated with /set-channel, any non-command message posted in that channel by a non-bot user is intercepted by the on_message handler in bot.py and routed to the POST /sessions/{id}/narrate endpoint as a player action.
The API response is rendered as a rich Discord Embed containing:
? The AI narration text (up to 2000 characters, paginated if longer)
? A ? Choices ? field listing the 3 suggested player actions
? An interactive discord.ui.View with reaction-style buttons for quick action routing
? A footer crediting the acting player and showing the Storyteller AI watermark

7.9 Rate Limiting & Cost Management

Uncontrolled use of GPT-4o can generate substantial API costs at scale. The rate limiter enforces two independent safeguards per guild: a per-day token budget and a per-message cooldown window. Both are tracked in Redis with TTL-based keys.

ControlMechanismDefaultConfig KeyDaily Token BudgetRedis counter with 24h TTL, per guild ID200,000 tokens/dayDAILY_TOKEN_BUDGET_PER_GUILDNarration CooldownRedis key with TTL, per guild+channel30 secondsNARRATION_COOLDOWN_SECONDSContext SummarizationAuto-summarize events older than 10 turns to reduce prompt token count10 events maxHardcoded in state engineEmbedding ModelUse text-embedding-3-small (much cheaper than large)$0.02/1M tokensOPENAI_EMBEDDING_MODEL

? COST PROJECTION NOTE
At average prompt sizes of ~3,000 tokens input + ~700 tokens output per narration call, with GPT-4o pricing of approximately $5/1M input tokens and $15/1M output tokens (as of mid-2025), a session of 20 narration calls costs roughly $0.42. A guild running 5 sessions/week would spend approximately $8–10/month in LLM costs. Budget accordingly and consider offering tiered access.

Appendix B — Testing Strategy
The test suite is organized into three tiers: unit tests for isolated logic, integration tests for pipeline and route behavior, and mock-LLM tests for route responses that would otherwise incur API costs. All tests use pytest with pytest-asyncio and pytest-mock.
Unit Tests — State Machine

python — tests/test_state_machine.py
"""Unit tests for StorytellerStateMachine transitions and meter logic."""
import pytest

from app.core.state_machine import StorytellerStateMachine
from app.core.state_machine_models import SessionState
from app.enums import StateMachineState


@pytest.fixture
def default_state():
    return SessionState(tension_level=5, doom_clock=0, momentum=0)


@pytest.fixture
def machine(default_state):
    return StorytellerStateMachine(state=default_state)


def test_valid_transition_scene_open_to_player_acting(machine):
    result = machine.transition_to(StateMachineState.PLAYER_ACTING)
    assert result is True
    assert machine.state.machine_state == StateMachineState.PLAYER_ACTING


def test_invalid_transition_scene_open_to_combat_without_guard(machine):
    """Cannot jump directly from SCENE_OPEN to COMBAT — must go through PLAYER_ACTING."""
    result = machine.transition_to(StateMachineState.COMBAT)
    assert result is False  # COMBAT not in SCENE_OPEN valid transitions


def test_tension_clamping(machine):
    machine.update_tension(100)
    assert machine.state.tension_level == 10

    machine.update_tension(-100)
    assert machine.state.tension_level == 1


def test_doom_clock_advances(machine):
    machine.advance_doom(5)
    assert machine.state.doom_clock == 5


def test_doom_event_triggers_at_12(machine):
    machine.state.doom_clock = 11
    result = machine.advance_doom(1)
    assert result == 12  # Returns 12 briefly before event fires
    # After event, doom resets to 8
    assert machine.state.doom_clock == 8


def test_serialize_deserialize_roundtrip(machine):
    machine.update_tension(3)
    machine.advance_doom(2)
    serialized = machine.serialize()

    restored = StorytellerStateMachine.deserialize(serialized)
    assert restored.state.tension_level == machine.state.tension_level
    assert restored.state.doom_clock == machine.state.doom_clock

import pytest

from unittest.mock import MagicMock, patch

from app.ingestion.pdf_parser import WhiteWolfPDFParser
from app.ingestion.chunk_embedder import classify_chunk, detect_splat_relevance

def test_classify_chunk_rule():
    text = "Roll Strength + Brawl at difficulty 6. Each success inflicts one health level."
    assert classify_chunk(text) == LoreCategory.RULE


def test_classify_chunk_power():
    text = "Discipline: Dominate •• — Mesmerism. The Kindred locks eyes and speaks one command."
    assert classify_chunk(text) == LoreCategory.POWER

    text = "The Camarilla was founded in 1493 in response to the Anarch Revolt."
    assert classify_chunk(text) == LoreCategory.LORE

    text = "The history of this place stretches back centuries."
    splats = detect_splat_relevance(text, None)
    assert splats == ["universal"]

"""Route tests using mock LLM client to avoid API costs."""
import pytest
import pytest_asyncio
from httpx import AsyncClient, ASGITransport
from unittest.mock import AsyncMock, MagicMock, patch

from app.main import app


@pytest_asyncio.fixture
async def client():
    async with AsyncClient(
        transport=ASGITransport(app=app),
        base_url="http://test",
    ) as ac:
        yield ac


@pytest.mark.asyncio
async def test_health_check(client):
    response = await client.get("/health")
    assert response.status_code == 200
    assert response.json()["status"] == "alive"


@pytest.mark.asyncio
async def test_create_session(client):
    payload = {"name": "Test Chronicle", "mode": "solo", "setting": "vampire"}

    with patch("app.db.crud.create_game_session") as mock_create:
        mock_session = MagicMock()
        mock_session.id = "00000000-0000-0000-0000-000000000001"
        mock_session.name = "Test Chronicle"
        mock_session.mode.value = "solo"
        mock_session.setting.value = "vampire"
        mock_session.is_active = True
        mock_session.state_json = {}
        mock_create.return_value = mock_session

        response = await client.post("/api/v1/sessions/", json=payload)
        assert response.status_code == 201
Appendix C — Roadmap & Future Enhancements
The Storyteller AI as described in this document represents a solid, production-ready v1.0 foundation. The following enhancements are planned or proposed for future versions, roughly ordered by implementation complexity.

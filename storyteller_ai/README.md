Storyteller AI Backend
======================
Generated by bootstrap_backend.py

Quickstart
----------
1. Run `install_dependencies.bat` once.
2. The installer creates a desktop shortcut for `launch_storyteller_ai.bat` automatically.
3. Double-click the desktop shortcut or `launch_storyteller_ai.bat` to start the backend.
4. Open your browser at:

    http://127.0.0.1:8000/

4. Use the frontend UI or the `/sessions` and `/gm` endpoints.

Local LLM Support
-----------------
Set `LLM_PROVIDER` to one of:

- `openai` — uses `OPENAI_API_KEY`
- `anthropic` — uses `ANTHROPIC_API_KEY`
- `ollama` — uses a local Ollama HTTP endpoint
- `mock` — uses a packaged offline mock provider

Example for Ollama:

    set LLM_PROVIDER=ollama
    set OLLAMA_URL=http://127.0.0.1:11434
    set OLLAMA_MODEL=llama2
    uvicorn backend.main:app --reload

Example for mock mode:

    set LLM_PROVIDER=mock
    uvicorn backend.main:app --reload

Frontend
--------
Start the backend and open the UI in a browser at:

    http://127.0.0.1:8000/

The frontend includes:
- session creation
- chat / GM step input
- PDF upload for document ingestion
- document retrieval query

Document ingestion API
----------------------
POST /documents/upload
- form field: `file` (PDF file)

GET /documents/list
- returns loaded documents and sizes

POST /documents/retrieve
- request body: `{ "query": "..." }`
- returns the best matching text from uploaded documents

Endpoints
---------
POST /sessions/create
- Request body: `{ "mode": "group" }`
- Response: `{ "session_id": "...", "mode": "group" }`

POST /gm/step
- Request body: `{ "session_id": "...", "message": "Your action here" }`
- Response: `{ "text": "...", "mode": "group" }`

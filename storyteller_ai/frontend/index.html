Storyteller AI

How to use this page

  1. Click Create Game to start a new story session.
  2. Copy the session ID into the Session ID field if it is not already filled.
  3. Type your next story action or scene and click Continue Story.
  4. Upload PDFs to teach the AI about characters, locations, or world lore.
  5. Ask questions using the retrieve panel to reference uploaded documents.
The page is arranged in order: game setup, story continuation, document upload, then document retrieval.
Tip: Use a setup prompt like "The heroes arrive at a ruined castle and hear footsteps in the dark hallway."
